# 趋势资金事件驱动策略 — 设计文档

## 一、策略概述

**策略名称**: 趋势资金事件驱动策略  
**策略类型**: 事件驱动型选股策略  
**目标用户**: 个人投资者，50万资金，3-5只持仓，日频交易

### 1.1 核心逻辑

趋势资金是指当日成交量显著高于近期平均水平的资金，其持续入场往往预示着股价上涨趋势的形成。

**信号类型**：
| 信号 | 条件 | 操作 |
|------|------|------|
| 强势买入 | 趋势资金连续入场 + 价格上涨 | 买入 |
| 预警观察 | 趋势资金入场但价格下跌 | 观察 |
| 卖出 | 趋势资金连续流出 或 放量滞涨 | 卖出 |

### 1.2 数据来源

| 数据 | 来源 | 用途 |
|------|------|------|
| 股票池 | 自选股 (SQLite watchlist) | 选股范围 |
| 分钟K线 | ClickHouse (klines_minute) | 计算趋势资金强度 |
| 日K线 | ClickHouse (klines_daily) | 辅助验证趋势 |
| 股票信息 | SQLite (Stock表) | 过滤ST/停牌/市值 |
| 实时行情 | QMT | 行业分类等 |

### 1.3 过滤条件

- 排除ST股、停牌股
- 市值 >= 10亿元
- 行业：科技类（电子、计算机、通信、传媒等）

## 二、因子设计

### 2.1 趋势资金强度因子

```python
# 基于分钟K线计算
当日累计成交量 = sum(分钟volume)
近20日平均分钟成交量 = avg(daily_volume) / 日内分钟数
趋势强度 = 当日累计成交量 / 近20日平均分钟成交量
```

### 2.2 连续性因子

```python
# 趋势资金是否持续放大
趋势连续性 = 趋势强度(t) * 0.5 + 趋势强度(t-1) * 0.3 + 趋势强度(t-2) * 0.2
```

### 2.3 动量因子

```python
# 当日涨跌幅
动量 = (收盘价 - 开盘价) / 开盘价 * 100
```

### 2.4 综合信号分

```
综合信号分 = 趋势强度 × 0.4 + 连续性 × 0.3 + 动量 × 0.3
```

## 四、策略流程

```
每日 15:05 收盘后:
  1. 读取自选股列表
  2. 获取每只股当日分钟K线 + 近20日日K线
  3. 计算趋势资金因子:
     - 当日累计分钟成交量
     - 近20日平均成交量
     - 趋势强度 = 当日 / 平均
     - 趋势连续性 (3日加权)
     - 动量 (当日涨跌幅)
  4. 过滤:
     - ST/停牌: is_st=0, is_suspend=0
     - 市值: total_mv >= 10亿
     - 行业: 科技类 (电子/计算机/通信/传媒/软件等)
  5. 排序: 按综合信号分降序
  6. 取 Top5
  7. 生成信号
  8. 存入数据库
  9. 返回前端展示
```
